# 困惑点审计

交付前快速过一遍：

## 1. 对象

- 每个符号对应什么真实对象？
- 向量的每个分量是什么？
- 矩阵的行和列分别是什么？
- 概率、损失、指标各自在测什么？

## 2. 数字

- 这些数字从哪里来？
- 它们是编码出来的、训练学出来的，还是人工设的例子？
- 正负号和大小各是什么意思？
- 偏置有没有被解释成门槛/基线？

## 3. 例子

- 公式前有没有原始例子？
- 读者能否看见：
  原始数据 -> 特征 -> 隐藏量 -> 输出？
- 有没有至少一个相近例子或反例？

## 4. 推导

- 每个公式有没有回答一个明确问题？
- 有没有先讲一步，再讲一般式？
- 链式法则里直接路径和间接路径有没有分开？
- 有没有哪一步初学者会本能问“为什么可以这样写”？

## 5. 追问

- 这个量从哪里来？
- 为什么是这个公式？
- 这个负号是什么意思？
- 这个数值大小是什么意思？
- 这个式子没有在说什么？
